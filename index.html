Nauka Programowania: HTML & CSS

Podstawy HTML

  • HTML to język znaczników, który tworzy strukturę strony.
  • Każdy element to znacznik, np. <p>Akapit</p>
  • Znaczniki mogą mieć atrybuty, np. <img src="..." alt="...">
  • Struktura strony: 
    <!DOCTYPE html>
<html>
  <head>
    <title>Moja strona</title>
  </head>
  <body>
    Tutaj jest treść!
  </body>
</html>
  • Najważniejsze znaczniki:
    • <h1> do <h6> – nagłówki
    • <p> – akapit
    • <ul>, <ol>, <li> – listy
    • <a> – link
    • <img> – obrazek
    • <table> – tabela

Podstawy CSS

  • CSS pozwala stylować stronę: kolory, czcionki, marginesy, animacje.
  • Dodajemy w <style> w <head> lub w osobnym pliku .css.
  • Selektory CSS wybierają elementy, np. h1 { color: blue; }
  • Przykład: 
    <style>
  h1 {
    color: #e17055;
    text-align: center;
  }
  p {
    color: #2a7ae2;
    font-size: 1.2em;
  }
</style>
  • Możesz stylować dowolny element!

Animacje w CSS

  • Animacje pozwalają na płynne zmiany stylu elementów.
  • Używamy transition i @keyframes.
  • Przykład animowanego przycisku: 
    button {
  transition: background 0.3s, transform 0.3s;
}
button:hover {
  background: #e17055;
  transform: scale(1.08) rotate(-2deg);
}
  • Przykład animacji boxa: 
    @keyframes bouncebox {
  0%, 100% { transform: translateY(0);}
  50% { transform: translateY(-10px);}
}
.anim-demo {
  animation: bouncebox 2s infinite;
}
Animacja!

Ćwiczenia praktyczne

  • Stwórz nagłówek <h1>Moja pierwsza strona</h1>
  • Dodaj akapit <p>To jest mój pierwszy tekst.</p>
  • Dodaj listę hobby
  • Dodaj obrazek z internetu
  • Dodaj link do ulubionej strony
  • Stwórz tabelę z planem lekcji
  • Dodaj własny styl CSS
  • Zrób animowany przycisk
Przykład HTML: 
<h1>Witaj na mojej stronie!</h1>
<p>To jest przykładowy akapit.</p>
<img src="https://picsum.photos/200" alt="Obrazek">

h1 {
  color: #e17055;
  text-align: center;
}
img {
  border-radius: 12px;
  box-shadow: 0 4px 16px #2a7ae244;

button {
  transition: background 0.3s, transform 0.3s;
}
button:hover {
  background: #e17055;
  transform: scale(1.08) rotate(-2deg);
}
